# TypeScript

대표 기능

- 엄격한 타입관리
- 제너릭
- 인터페이스

## 초기설정

#### 타입스크립트 설치

```
npm i -g typescript
```

```
tsc --init
```

#### TSLint

```
npm install tslint typescript -g
```

```
tslint --init
tslint -c tslint.json 'src/\*_/_.ts'
```

## Airbnb 스타일

npm i tslint-config-airbnb

## Google TypeScript Style

```
npx gts init
```

## TSDX

```
npx tsdx create <프로젝트_이름>
```
