## 얕은 복사

원본 값과 복사된 값이 같은 값을 참조

```js
let origin = { name: 'kkm' };
let copy = origin;

copy.name = 'minomad';

console.log(origin.name); // minomad
console.log(origin === copy); // true
```

#### for ~ in

```js
let messenger = {
  name: 'kakao talk',
  manufacture: 'kakao',
};

const cloneObject = {};

for (const key in messenger) {
  cloneObject[key] = messenger[key];
}
```
